🔍 Audit pré-lancement complet — My Language Coach

06/07/2026 · 7 audits parallèles ancrés dans le code (Produit/UX · Bugs/QA · Sécurité · Infra · Qualité IA · Monétisation · Stores/Marketing) · chaque constat vérifié dans le code source avec preuve fichier:ligne · fiches stores et site inspectés en live · s'appuie sur l'audit du 27/06/2026
Verdict : le produit est bon — bien meilleur que son économie, sa mesurabilité et sa présence sur les stores. Ne dépense pas un euro en pubs pour l'instant. La boucle cœur est soignée et différenciante (Lisa persistante + mémoire inter-sessions = un avantage qu'aucun concurrent majeur n'a), la sécurité est réellement solide, et la plupart des P0 du 27/06 ont été corrigés. Quatre blocs bloquent le lancement — tous fermables dans les ~8 semaines avant la fenêtre naturelle de début septembre :
  1. L'économie unitaire est à l'envers, par accident. Deux décisions déjà prises — TTS pas cher pour les gratuits, « Coach's voice = Pro » — n'ont jamais été câblées. ElevenLabs (coût 5–10×) sert tout le monde ; un utilisateur gratuit typique coûte 4–10 $/mois et même l'abonné Pro moyen est à marge négative. En plus : le décompte de quota fait confiance à une valeur envoyée par le client, et le webhook RevenueCat retire le Pro dès que l'auto-renouvellement est désactivé.
  2. Tu voles à l'aveugle. PostHog jamais initialisé, aucun MMP/SKAdNetwork/pixel, les grilles tarifaires Deepgram sous-estiment les coûts 10×, dashboard admin en 500, zéro alerte.
  3. Deux bugs P0 de corruption de données livrés avec la conversation continue : /end sur le fil infini corrompt les streaks et envoie tout le transcript à OpenAI ; le calcul de checkpoint compte les jours d'inactivité comme du temps de pratique.
  4. Les fiches stores notées ~29/100. Pas de sous-titre iOS, catégorie Play = « Outils », zéro note, pas de vidéo, métadonnées FR en anglais, captures d'écran vides sur connexion faible.
